# Cognitive Prep: Student Study Platform

A premium, distraction-free study environment combining an AI subject tutor, a cognitive mentor, Pomodoro workflow integration, and real-time audio and speech utilities.

Built with **pure HTML, CSS, and Vanilla JavaScript**. No compilers, builders, or framework setups required.

## Key Features

- **Fast-Stream Chat (Rapid Inquiry)**: Engage directly with a friendly, interactive tutor utilizing the Google Gemini 3.5 Flash model.
- **Recall & Synthesis (Deep Memory)**: Consolidate learning sessions and synthesize a comprehensive "Recall Mastery" guide, featuring isolated keywords and a closed-eyes recall check.
- **Integrated Pomodoro Timer**: Switch between work and break intervals with visually anchored progress indicator lines and audio notifications.
- **Focus Mode**: Transition into a full-screen mode instantly to eliminate screen distractions.
- **Voice Typing**: Leverage native Speech Recognition to transcribe notes and inputs hands-free.
- **Session Progress Chart**: Custom dynamic SVG rendering tracks interaction activity metrics seamlessly.

## Quick Start

1. Open `index.html` directly in any modern web browser.
2. Enter your **Gemini API Key** in the overlay modal.
3. Start studying!
